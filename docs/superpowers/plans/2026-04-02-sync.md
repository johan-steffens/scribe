# Sync Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add opt-in state sync across machines via a pluggable provider system (GitHub Gist, S3, iCloud Drive, JSONBin, Dropbox, self-hosted REST, or a custom file path), with secrets stored in the OS keychain, merge-on-pull conflict resolution, and automatic syncing via the background daemon.

**Architecture:** A new `sync` Cargo feature gates all sync code. A `SyncProvider` async trait abstracts the remote backend; `SyncEngine` holds a boxed provider and runs the pull→merge→push cycle. The daemon loop gets a second ticker for sync; when the REST master role is configured, the daemon also spawns an axum HTTP server. Secrets never appear in `config.toml` — they live exclusively in the OS keychain via the `keyring` crate.

**Tech Stack:** `reqwest` (HTTP clients), `keyring` (OS keychain), `axum` + `tokio` (REST master server), `uuid` (machine ID), `async-trait` (object-safe async trait), `hex` (secret encoding), `sha2` (snapshot content hash for idempotency check).

```rust
//! `StateSnapshot` — the full serialisable state document pushed to and pulled
//! from the remote provider.

use chrono::{DateTime, Utc};
use serde::{Deserialize, Serialize};
use sha2::{Digest, Sha256};
use uuid::Uuid;

use crate::domain::{CaptureItem, Project, Reminder, Task, TimeEntry, Todo};

/// The full Scribe state, serialised for remote storage.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct StateSnapshot {
    /// UTC timestamp at which this snapshot was produced.
    pub snapshot_at: DateTime<Utc>,
    /// Unique identifier for the machine that produced this snapshot.
    pub machine_id: Uuid,
    /// Incremented on breaking changes to the snapshot schema.
    pub schema_version: u32,
    /// All projects (including archived).
    pub projects: Vec<Project>,
    /// All tasks (including archived).
    pub tasks: Vec<Task>,
    /// All todos (including archived).
    pub todos: Vec<Todo>,
    /// All time entries (including archived).
    pub time_entries: Vec<TimeEntry>,
    /// All reminders (including archived).
    pub reminders: Vec<Reminder>,
    /// All capture items (including processed).
    pub capture_items: Vec<CaptureItem>,
}

impl StateSnapshot {
    /// Current schema version. Bump when the snapshot shape changes in a
    /// breaking way.
    pub const SCHEMA_VERSION: u32 = 1;

    /// Returns a hex-encoded SHA-256 hash of the snapshot contents.
    ///
    /// `snapshot_at` and `machine_id` are excluded so the hash reflects data
    /// content only, not metadata. Used for idempotency: if the hash matches
    /// the remote after merging, the push is skipped.
    #[must_use]
    pub fn content_hash(&self) -> String {
        // Serialise only the data fields, not the metadata fields.
        #[derive(Serialize)]
        struct HashableSnapshot<'a> {
            schema_version: u32,
            projects: &'a Vec<Project>,
            tasks: &'a Vec<Task>,
            todos: &'a Vec<Todo>,
            time_entries: &'a Vec<TimeEntry>,
            reminders: &'a Vec<Reminder>,
            capture_items: &'a Vec<CaptureItem>,
        }
        let hashable = HashableSnapshot {
            schema_version: self.schema_version,
            projects: &self.projects,
            tasks: &self.tasks,
            todos: &self.todos,
            time_entries: &self.time_entries,
            reminders: &self.reminders,
            capture_items: &self.capture_items,
        };
        let json = serde_json::to_vec(&hashable).expect("StateSnapshot is always serialisable");
        let digest = Sha256::digest(&json);
        hex::encode(digest)
    }
}
```

```rust
//! Sync feature — provider trait, engine, and snapshot types.

pub mod snapshot;

use async_trait::async_trait;
use thiserror::Error;

pub use snapshot::StateSnapshot;

/// Errors that can occur during a sync operation.
#[derive(Debug, Error)]
pub enum SyncError {
    /// The remote resource does not exist yet (first push not completed).
    #[error("not found: {0}")]
    NotFound(String),
    /// Authentication with the provider failed.
    #[error("auth error: {0}")]
    Auth(String),
    /// A network or I/O error occurred.
    #[error("transport error: {0}")]
    Transport(String),
    /// The remote snapshot cannot be deserialised.
    #[error("invalid snapshot: {0}")]
    InvalidSnapshot(String),
    /// The keychain is unavailable or the secret is missing.
    #[error("keychain error: {0}")]
    Keychain(String),
    /// Any other provider-specific error.
    #[error("{0}")]
    Other(String),
}

/// Abstraction over a remote sync backend.
///
/// Implementors push and pull a [`StateSnapshot`] to/from a remote location.
/// The [`SyncEngine`] holds a `Box<dyn SyncProvider>` and is provider-agnostic.
#[async_trait]
pub trait SyncProvider: Send + Sync {
    /// Upload the local state snapshot to the remote.
    ///
    /// # Errors
    ///
    /// Returns a [`SyncError`] if the push fails for any reason.
    async fn push(&self, snapshot: &StateSnapshot) -> Result<(), SyncError>;

    /// Download the current remote snapshot.
    ///
    /// Returns [`SyncError::NotFound`] when no remote state exists yet (first
    /// run on a new machine before any push has occurred).
    ///
    /// # Errors
    ///
    /// Returns a [`SyncError`] if the pull fails for any reason.
    async fn pull(&self) -> Result<StateSnapshot, SyncError>;
}
```

//! OS keychain abstraction for storing and retrieving sync secrets.
//!
//! Secrets are stored under service names of the form
//! `scribe.sync.<provider>.<field>`. The `keyring` crate provides a unified
//! API across macOS Keychain, Linux libsecret, and Windows Credential Manager.

use keyring::Entry;

use crate::sync::SyncError;

/// Reads and writes sync secrets in the OS keychain.
#[derive(Debug)]
pub struct KeychainStore;

impl KeychainStore {
    /// Returns the canonical service name for a provider field.
    ///
    /// Format: `scribe.sync.<provider>.<field>`.
    ///
    /// # Examples
    ///
    /// ```
    /// use scribe::sync::keychain::KeychainStore;
    /// assert_eq!(KeychainStore::service_name("gist", "token"), "scribe.sync.gist.token");
    /// ```
    #[must_use]
    pub fn service_name(provider: &str, field: &str) -> String {
        format!("scribe.sync.{provider}.{field}")
    }

    /// Retrieves a secret from the OS keychain.
    ///
    /// `provider` and `field` form the service name (e.g. `"gist"`, `"token"`).
    /// The username stored in the entry is always `"scribe"`.
    ///
    /// # Errors
    ///
    /// Returns [`SyncError::Keychain`] if the secret is missing or the keychain
    /// daemon is unavailable.
    pub fn get(provider: &str, field: &str) -> Result<String, SyncError> {
        let service = Self::service_name(provider, field);
        let entry = Entry::new(&service, "scribe").map_err(|e| {
            SyncError::Keychain(format!(
                "sync requires a keychain daemon to store secrets securely. \
                 Install and start gnome-keyring or kwallet, then re-run \
                 `scribe sync configure`. (detail: {e})"
            ))
        })?;
        entry.get_password().map_err(|e| {
            SyncError::Keychain(format!(
                "secret '{service}' not found in keychain — run \
                 `scribe sync configure` to set it up. (detail: {e})"
            ))
        })
    }

    /// Stores a secret in the OS keychain.
    ///
    /// # Errors
    ///
    /// Returns [`SyncError::Keychain`] if the keychain is unavailable.
    pub fn set(provider: &str, field: &str, secret: &str) -> Result<(), SyncError> {
        let service = Self::service_name(provider, field);
        let entry = Entry::new(&service, "scribe").map_err(|e| {
            SyncError::Keychain(format!(

```rust
//!
//! Useful for custom file paths managed by a third-party sync daemon such as
//! Dropbox Desktop, OneDrive, or Syncthing.

use std::path::PathBuf;

use async_trait::async_trait;

use crate::sync::{StateSnapshot, SyncError, SyncProvider};

/// Syncs state via a plain JSON file at a configured path.
#[derive(Debug)]
pub struct FileProvider {
    path: PathBuf,
}

impl FileProvider {
    /// Creates a new `FileProvider` targeting the given path.
    #[must_use]
    pub fn new(path: PathBuf) -> Self {
        Self { path }
    }
}

#[async_trait]
impl SyncProvider for FileProvider {
    async fn push(&self, snapshot: &StateSnapshot) -> Result<(), SyncError> {
        if let Some(parent) = self.path.parent() {
            std::fs::create_dir_all(parent)
                .map_err(|e| SyncError::Transport(format!("could not create directory: {e}")))?;
        }
        let json = serde_json::to_string_pretty(snapshot)
            .map_err(|e| SyncError::Other(format!("serialisation failed: {e}")))?;
        std::fs::write(&self.path, json)
            .map_err(|e| SyncError::Transport(format!("write failed: {e}")))?;
        Ok(())
    }

    async fn pull(&self) -> Result<StateSnapshot, SyncError> {
        if !self.path.exists() {
            return Err(SyncError::NotFound(format!(
                "no state file at '{}'", self.path.display()
            )));
        }
        let json = std::fs::read_to_string(&self.path)
            .map_err(|e| SyncError::Transport(format!("read failed: {e}")))?;
        serde_json::from_str(&json)
            .map_err(|e| SyncError::InvalidSnapshot(format!("deserialisation failed: {e}")))
    }
}
```

```rust
//! `SyncEngine` — pull → merge → push sync cycle.
//!
//! The engine is provider-agnostic: it holds a `Box<dyn SyncProvider>` and
//! delegates all remote I/O to it. The merge algorithm is field-level
//! last-write-wins keyed on entity `slug`.

use std::path::PathBuf;

use chrono::Utc;
use serde::{Deserialize, Serialize};

use crate::sync::{StateSnapshot, SyncError, SyncProvider};

// ── sync state file ────────────────────────────────────────────────────────

/// Persisted record of the last sync attempt, stored at
/// `~/.local/share/scribe/sync-state.json`.
#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct SyncState {
    /// UTC timestamp of the last successful sync, if any.
    pub last_sync_at: Option<chrono::DateTime<Utc>>,
    /// Error message from the last failed sync, if any.
    pub last_error: Option<String>,
    /// Name of the provider used for the last sync.
    pub provider: Option<String>,
}

impl SyncState {
    /// Loads the sync state from disk, returning a default if absent.
    #[must_use]
    pub fn load(path: &PathBuf) -> Self {
        std::fs::read_to_string(path)
            .ok()
            .and_then(|s| serde_json::from_str(&s).ok())
            .unwrap_or_default()
    }

    /// Persists the sync state to disk.
    pub fn save(&self, path: &PathBuf) -> anyhow::Result<()> {
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent)?;
        }
        let json = serde_json::to_string_pretty(self)?;
        std::fs::write(path, json)?;
        Ok(())
    }
}

// ── engine ─────────────────────────────────────────────────────────────────

/// Orchestrates the pull → merge → push sync cycle.
pub struct SyncEngine {
    provider: Box<dyn SyncProvider>,
    sync_state_path: PathBuf,
    provider_name: String,
}

impl std::fmt::Debug for SyncEngine {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        f.debug_struct("SyncEngine")
            .field("provider_name", &self.provider_name)
            .finish_non_exhaustive()
    }
}

impl SyncEngine {
    /// Creates a new `SyncEngine` wrapping the given provider.
    #[must_use]
    pub fn new(
        provider: Box<dyn SyncProvider>,
        sync_state_path: PathBuf,
        provider_name: String,
    ) -> Self {
        Self { provider, sync_state_path, provider_name }
    }

    /// Runs one full pull → merge → push cycle against the remote.
    ///
    /// If the remote has no state yet ([`SyncError::NotFound`]), the local
    /// state is pushed as the initial snapshot. If the merged snapshot is
    /// content-identical to the remote, the push is skipped.
    ///
    /// # Errors
    ///
    /// Returns a [`SyncError`] if the pull or push fails for a reason other
    /// than `NotFound`.
    pub async fn run_once(&self, local: StateSnapshot) -> Result<StateSnapshot, SyncError> {
        let merged = match self.provider.pull().await {
            Ok(remote) => {
                let mut merged = local;
                Self::merge_into(&mut merged, &remote);
                merged
            }
            Err(SyncError::NotFound(_)) => {
                // First sync from this machine — push local state as the seed.
                local
            }
            Err(e) => return Err(e),
        };

        // Skip push if nothing changed.
        let remote_after_merge = self.provider.pull().await;
        let should_push = match remote_after_merge {
            Ok(ref remote) => remote.content_hash() != merged.content_hash(),
            Err(SyncError::NotFound(_)) => true,
            Err(e) => return Err(e),
        };

        if should_push {
            self.provider.push(&merged).await?;
        }

        Ok(merged)
    }

    /// Merges `remote` into `local` in place.
    ///
    /// For each entity type, entities are keyed by `slug`. The merge rule is:
    /// - Remote-only entity → insert into local.
    /// - Local-only entity → keep unchanged.
    /// - Both exist, remote newer (`remote.updated_at > local.updated_at`) → replace local.
    /// - Both exist, local newer or equal → keep local.
    pub fn merge_into(local: &mut StateSnapshot, remote: &StateSnapshot) {
        merge_vec(&mut local.projects, &remote.projects, |e| &e.slug, |e| e.updated_at);
        merge_vec(&mut local.tasks, &remote.tasks, |e| &e.slug, |e| e.updated_at);
        merge_vec(&mut local.todos, &remote.todos, |e| &e.slug, |e| e.updated_at);
        merge_vec(&mut local.time_entries, &remote.time_entries, |e| &e.slug, |e| e.created_at);
        merge_vec(&mut local.reminders, &remote.reminders, |e| &e.slug, |e| e.created_at);
        merge_vec(&mut local.capture_items, &remote.capture_items, |e| &e.slug, |e| e.created_at);
    }
}

// ── merge helper ───────────────────────────────────────────────────────────

fn merge_vec<T, S, F>(local: &mut Vec<T>, remote: &[T], slug_fn: S, ts_fn: F)
where
    T: Clone,
    S: Fn(&T) -> &String,
    F: Fn(&T) -> chrono::DateTime<Utc>,
{
    use std::collections::HashMap;

    // Index local by slug for O(1) lookup.
    let mut local_map: HashMap<String, usize> = local
        .iter()
        .enumerate()
        .map(|(i, e)| (slug_fn(e).clone(), i))
        .collect();

    for remote_entity in remote {
        let slug = slug_fn(remote_entity).clone();
        match local_map.get(&slug) {
            Some(&idx) => {
                // Both exist — replace if remote is strictly newer.
                if ts_fn(remote_entity) > ts_fn(&local[idx]) {
                    local[idx] = remote_entity.clone();
                }
            }
            None => {
                // Remote-only — insert.
                let new_idx = local.len();
                local.push(remote_entity.clone());
                local_map.insert(slug, new_idx);
            }
        }
    }
}
```

```rust
//! `GistProvider` — syncs state via a secret GitHub Gist.
//!
//! On first push a new secret gist is created and the returned ID is written
//! back to `config.toml` via the caller. Subsequent pushes update the gist.

use async_trait::async_trait;
use reqwest::Client;
use serde::{Deserialize, Serialize};
use serde_json::json;

use crate::sync::keychain::KeychainStore;
use crate::sync::{StateSnapshot, SyncError, SyncProvider};

const GITHUB_API: &str = "https://api.github.com";
const FILE_NAME: &str = "scribe-state.json";

/// Syncs state via a secret GitHub Gist.
#[derive(Debug)]
pub struct GistProvider {
    /// Existing gist ID, or `None` if not created yet.
    pub gist_id: Option<String>,
    client: Client,
}

#[derive(Debug, Deserialize)]
struct GistFile {
    content: String,
}

#[derive(Debug, Deserialize)]
struct GistResponse {
    id: String,
    files: std::collections::HashMap<String, GistFile>,
}

impl GistProvider {
    /// Creates a new `GistProvider`.
    ///
    /// `gist_id` is `None` on first use and populated after the initial push.
    pub fn new(gist_id: Option<String>) -> Result<Self, SyncError> {
        let client = Client::builder()
            .user_agent("scribe-sync/1.0")
            .build()
            .map_err(|e| SyncError::Transport(format!("failed to build HTTP client: {e}")))?;
        Ok(Self { gist_id, client })
    }

    fn token() -> Result<String, SyncError> {
        KeychainStore::get("gist", "token")
    }
}

#[async_trait]
impl SyncProvider for GistProvider {
    async fn push(&self, snapshot: &StateSnapshot) -> Result<(), SyncError> {
        let token = Self::token()?;
        let content = serde_json::to_string_pretty(snapshot)
            .map_err(|e| SyncError::Other(format!("serialise failed: {e}")))?;
        let body = json!({
            "description": "Scribe state sync",
            "public": false,
            "files": { FILE_NAME: { "content": content } }
        });

        let (method, url) = match &self.gist_id {
            Some(id) => ("PATCH", format!("{GITHUB_API}/gists/{id}")),
            None => ("POST", format!("{GITHUB_API}/gists")),
        };

        let req = if method == "PATCH" {
            self.client.patch(&url)
        } else {
            self.client.post(&url)
        };

        let resp = req
            .bearer_auth(&token)
            .json(&body)
            .send()
            .await
            .map_err(|e| SyncError::Transport(format!("HTTP request failed: {e}")))?;

        if resp.status() == reqwest::StatusCode::UNAUTHORIZED {
            return Err(SyncError::Auth(
                "GitHub API returned 401 — check your token with `scribe sync configure`"
                    .to_owned(),
            ));
        }
        if !resp.status().is_success() {
            let status = resp.status();
            let body = resp.text().await.unwrap_or_default();
            return Err(SyncError::Transport(format!(
                "GitHub API returned {status}: {body}"
            )));
        }

        Ok(())
    }

    async fn pull(&self) -> Result<StateSnapshot, SyncError> {
        let id = self.gist_id.as_ref().ok_or_else(|| {
            SyncError::NotFound(
                "no gist_id configured — run `scribe sync configure` first".to_owned(),
            )
        })?;
        let token = Self::token()?;

        let resp = self
            .client
            .get(format!("{GITHUB_API}/gists/{id}"))
            .bearer_auth(&token)
            .send()
            .await
            .map_err(|e| SyncError::Transport(format!("HTTP request failed: {e}")))?;

        if resp.status() == reqwest::StatusCode::NOT_FOUND {
            return Err(SyncError::NotFound(format!("gist '{id}' not found")));
        }
        if resp.status() == reqwest::StatusCode::UNAUTHORIZED {
            return Err(SyncError::Auth(
                "GitHub API returned 401 — check your token with `scribe sync configure`"
                    .to_owned(),
            ));
        }
        if !resp.status().is_success() {
            let status = resp.status();
            let body = resp.text().await.unwrap_or_default();
            return Err(SyncError::Transport(format!("GitHub API returned {status}: {body}")));
        }

        let gist: GistResponse = resp
            .json()
            .await
            .map_err(|e| SyncError::InvalidSnapshot(format!("could not parse gist response: {e}")))?;

        let file = gist.files.get(FILE_NAME).ok_or_else(|| {
            SyncError::NotFound(format!("file '{FILE_NAME}' not found in gist"))
        })?;

        serde_json::from_str(&file.content)
            .map_err(|e| SyncError::InvalidSnapshot(format!("could not parse snapshot: {e}")))
    }
}
```

```rust
const API_BASE: &str = "https://api.jsonbin.io/v3";

/// Syncs state via JSONBin.io.
#[derive(Debug)]
pub struct JsonBinProvider {

}

impl JsonBinProvider {

    pub fn new(bin_id: Option<String>) -> Result<Self, SyncError> {
        let client = Client::builder()
            .build()
            .map_err(|e| SyncError::Transport(e.to_string()))?;

    }

    fn access_key() -> Result<String, SyncError> {
        KeychainStore::get("jsonbin", "access_key")
    }
}

#[async_trait]
impl SyncProvider for JsonBinProvider {
    async fn push(&self, snapshot: &StateSnapshot) -> Result<(), SyncError> {
        let key = Self::access_key()?;
        match &self.bin_id {
            Some(id) => {
                let resp = self.client
                    .put(format!("{API_BASE}/b/{id}"))
                    .header("X-Access-Key", &key)
                    .json(snapshot)
                    .send()
                    .await
                    .map_err(|e| SyncError::Transport(e.to_string()))?;
                if !resp.status().is_success() {
                    let status = resp.status();
                    return Err(SyncError::Transport(format!("JSONBin returned {status}")));
                }
            }
            None => {
                // Create new bin.
                let resp = self.client
                    .post(format!("{API_BASE}/b"))
                    .header("X-Access-Key", &key)
                    .header("X-Bin-Name", "scribe-state")
                    .header("X-Bin-Private", "true")
                    .json(snapshot)
                    .send()
                    .await
                    .map_err(|e| SyncError::Transport(e.to_string()))?;
                if !resp.status().is_success() {
                    let status = resp.status();
                    return Err(SyncError::Transport(format!("JSONBin create returned {status}")));
                }
                // The caller is responsible for reading `self.bin_id` after
                // the first push and persisting it to config.toml. This is
                // handled by `scribe sync configure` and the CLI sync command.
            }
        }
        Ok(())
    }

    async fn pull(&self) -> Result<StateSnapshot, SyncError> {
        let id = self.bin_id.as_ref().ok_or_else(|| {
            SyncError::NotFound("no bin_id — run `scribe sync configure` first".to_owned())
        })?;
        let key = Self::access_key()?;
        let resp = self.client
            .get(format!("{API_BASE}/b/{id}/latest"))
            .header("X-Access-Key", &key)
            .send()
            .await
            .map_err(|e| SyncError::Transport(e.to_string()))?;
        if resp.status() == reqwest::StatusCode::NOT_FOUND {
            return Err(SyncError::NotFound(format!("bin '{id}' not found")));
        }
        if !resp.status().is_success() {
            let status = resp.status();
            return Err(SyncError::Transport(format!("JSONBin returned {status}")));
        }
        // JSONBin wraps the response in `{ "record": {...}, "metadata": {...} }`.
        let wrapper: serde_json::Value = resp.json().await
            .map_err(|e| SyncError::InvalidSnapshot(e.to_string()))?;
        let record = wrapper.get("record")
            .ok_or_else(|| SyncError::InvalidSnapshot("missing 'record' field".to_owned()))?;
        serde_json::from_value(record.clone())
            .map_err(|e| SyncError::InvalidSnapshot(e.to_string()))
    }
```

```rust
//! Bearer token middleware for the REST master server.

use axum::{
    extract::Request,
    http::{HeaderMap, StatusCode},
    middleware::Next,
    response::Response,
};

/// Axum middleware that checks for a valid `Authorization: Bearer <secret>` header.
pub async fn require_bearer(
    headers: HeaderMap,
    request: Request,
    next: Next,
) -> Result<Response, StatusCode> {
    let expected = request
        .extensions()
        .get::<String>()
        .cloned()
        .unwrap_or_default();

    let provided = headers
        .get(axum::http::header::AUTHORIZATION)
        .and_then(|v| v.to_str().ok())
        .and_then(|v| v.strip_prefix("Bearer "))
        .unwrap_or("");

    if provided == expected {
        Ok(next.run(request).await)
    } else {
        Err(StatusCode::UNAUTHORIZED)
    }
}
```

```rust
//! REST master server for the self-hosted sync provider.
//!
//! Started by the daemon when `sync.rest.role = "master"`. Exposes
//! `GET /state` and `PUT /state`, both protected by a Bearer token.

pub mod auth;
pub mod handlers;

use std::sync::Arc;

use axum::{
    middleware,
    routing::{get, put},
    Router,
};
use tokio::{net::TcpListener, sync::RwLock, task::JoinHandle};

use crate::sync::snapshot::StateSnapshot;

use handlers::ServerState;

/// Starts the REST sync master server on the given port.
///
/// Pass `port = 0` to let the OS assign an available port (useful in tests).
/// Returns the bound port and a `JoinHandle` for the server task.
///
/// The server runs until the `JoinHandle` is dropped or aborted.
pub async fn start_server(
    port: u16,
    secret: String,
    initial_snapshot: StateSnapshot,
) -> (u16, JoinHandle<()>) {
    let state = ServerState {
        snapshot: Arc::new(RwLock::new(initial_snapshot)),
        secret: secret.clone(),
    };

    let app = Router::new()
        .route("/state", get(handlers::get_state))
        .route("/state", put(handlers::put_state))
        .layer(middleware::from_fn({
            let secret = secret.clone();
            move |headers, mut req: axum::extract::Request, next: middleware::Next| {
                req.extensions_mut().insert(secret.clone());
                auth::require_bearer(headers, req, next)
            }
        }))
        .with_state(state);

    let listener = TcpListener::bind(format!("0.0.0.0:{port}"))
        .await
        .expect("failed to bind TCP listener");

    let bound_port = listener.local_addr().expect("no local addr").port();

    let handle = tokio::spawn(async move {
        axum::serve(listener, app).await.expect("server error");
    });

    (bound_port, handle)
}
```

```rust
    let local_snap = StateSnapshot::from_db(conn)?;
    let sync_state_path = sync_state_path();
    let provider_name = format!("{:?}", config.sync.provider).to_lowercase();

    let engine = SyncEngine::new(provider, sync_state_path.clone(), provider_name.clone());

    let rt = tokio::runtime::Runtime::new()?;
    let result = rt.block_on(engine.run_once(local_snap));

    let mut state = SyncState::load(&sync_state_path);
    match result {
        Ok(merged) => {
            merged.write_to_db(conn)?;
            state.last_sync_at = Some(chrono::Utc::now());
            state.last_error = None;
            state.provider = Some(provider_name);
            state.save(&sync_state_path)?;
            println!("sync complete");
        }
        Err(e) => {
            state.last_error = Some(e.to_string());
            state.save(&sync_state_path)?;
            anyhow::bail!("sync failed: {e}");
        }
    }
    Ok(())
}
```

```rust
/// # Errors
///
/// Returns an error if any database write fails.
pub fn upsert_all(&self, projects: &[Project]) -> anyhow::Result<()> {
    let conn = self.conn.lock().unwrap();
    for p in projects {
        conn.execute(
            "INSERT INTO projects
                (slug, name, description, status, is_reserved, archived_at,
                 created_at, updated_at)
             VALUES (?1, ?2, ?3, ?4, ?5, ?6, ?7, ?8)
             ON CONFLICT(slug) DO UPDATE SET
                name        = excluded.name,
                description = excluded.description,
                status      = excluded.status,
                archived_at = excluded.archived_at,
                updated_at  = excluded.updated_at",
            rusqlite::params![
                p.slug,
                p.name,
                p.description,
                p.status.to_string(),
                p.is_reserved as i64,
                p.archived_at.map(|t| t.to_rfc3339()),
                p.created_at.to_rfc3339(),
                p.updated_at.to_rfc3339(),
            ],
        )?;
    }
```

```rust
// Rust guideline compliant 2026-02-21
//! `scribe daemon` — background reminder polling and optional sync loop.
//!
//! The daemon is a long-running process managed by the OS service supervisor
//! (launchd on macOS, systemd on Linux). It runs two independent tickers:
//!
//! 1. Reminder check — polls for due reminders every 30 s.
//! 2. Sync cycle — pushes/pulls state to the configured provider every
//!    `sync.interval_secs` (default 60 s). Only active when `sync.enabled = true`.
//!
//! When `sync.provider = "rest"` and `sync.rest.role = "master"`, the daemon
//! also starts an axum HTTP server on the configured port before entering the
//! loop.

use std::sync::{Arc, Mutex};
use std::thread;
use std::time::{Duration, Instant};

use rusqlite::Connection;

use crate::config::Config;
use crate::ops::ReminderOps;

const DEFAULT_REMINDER_INTERVAL_SECS: u64 = 30;

/// Launches the daemon loop, blocking until the process is interrupted.
///
/// # Errors
///
/// Returns an error only if the initial database read fails. Subsequent errors
/// are logged and do not abort the loop.
pub fn run(
    conn: Arc<Mutex<Connection>>,
    interval_secs: Option<u64>,
    config: &Config,
) -> anyhow::Result<()> {
    let reminder_interval =
        Duration::from_secs(interval_secs.unwrap_or(DEFAULT_REMINDER_INTERVAL_SECS));

    #[cfg(feature = "sync")]
    let sync_interval = Duration::from_secs(config.sync.interval_secs);

    #[cfg(feature = "sync")]
    let sync_enabled = config.sync.enabled;

    // Start REST master server if configured.
    #[cfg(feature = "sync")]
    {
        use crate::config::{RestRole, SyncProvider};
        if config.sync.enabled
            && config.sync.provider == SyncProvider::Rest
            && config.sync.rest.role == RestRole::Master
        {
            use crate::sync::{keychain::KeychainStore, snapshot::StateSnapshot};
            let secret = KeychainStore::get("rest", "secret")
                .map_err(|e| anyhow::anyhow!("could not read REST secret: {e}"))?;
            let port = config.sync.rest.port;
            let initial = StateSnapshot::from_db(&conn)?;
            let rt = tokio::runtime::Runtime::new()?;
            rt.spawn(async move {
                crate::server::start_server(port, secret, initial).await;
            });
            tracing::info!(port, "sync.rest.master.started");
            eprintln!("Scribe REST sync master listening on port {port}.");
        }
    }

    let reminder_ops = ReminderOps::new(Arc::clone(&conn));

    tracing::info!(
        reminder_interval_secs = reminder_interval.as_secs(),
        "daemon.start"
    );
    eprintln!(
        "Scribe daemon running — reminder poll every {}s. Press Ctrl-C to stop.",
        reminder_interval.as_secs()
    );

    let mut last_reminder_check = Instant::now() - reminder_interval;

    #[cfg(feature = "sync")]
    let mut last_sync = Instant::now() - sync_interval;

    loop {
        let now = Instant::now();

        // Reminder check.
        if now.duration_since(last_reminder_check) >= reminder_interval {
            match reminder_ops.check_due() {
                Ok(fired) => {
                    for reminder in &fired {
                        tracing::info!(
                            reminder.slug = %reminder.slug,
                            "daemon.reminder.fired"
                        );
                        crate::notify::fire(reminder);
                    }
                }
                Err(e) => {
                    tracing::warn!(error = %e, "daemon.reminder.poll.error");
                }
            }
            last_reminder_check = now;
        }

        // Sync cycle.
        #[cfg(feature = "sync")]
        if sync_enabled && now.duration_since(last_sync) >= sync_interval {
            run_sync_cycle(&conn, config);
            last_sync = now;
        }

        thread::sleep(Duration::from_secs(1));
    }
}

#[cfg(feature = "sync")]
fn run_sync_cycle(
    conn: &Arc<Mutex<Connection>>,
    config: &Config,
) {
    use crate::sync::{engine::SyncEngine, snapshot::StateSnapshot};
    use crate::sync::providers;

    let provider = match providers::from_config(config) {
        Ok(Some(p)) => p,
        Ok(None) => return,
        Err(e) => {
            tracing::warn!(error = %e, "daemon.sync.provider.error");
            return;
        }
    };

    let local = match StateSnapshot::from_db(conn) {
        Ok(s) => s,
        Err(e) => {
            tracing::warn!(error = %e, "daemon.sync.snapshot.error");
            return;
        }
    };

    let sync_state_path = {
        if let Some(dirs) = directories::ProjectDirs::from("", "", "scribe") {
            dirs.data_dir().join("sync-state.json")
        } else {
            std::path::PathBuf::from(".local/share/scribe/sync-state.json")
        }
    };
    let provider_name = format!("{:?}", config.sync.provider).to_lowercase();
    let engine = SyncEngine::new(provider, sync_state_path.clone(), provider_name.clone());

    let rt = match tokio::runtime::Runtime::new() {
        Ok(r) => r,
        Err(e) => {
            tracing::warn!(error = %e, "daemon.sync.runtime.error");
            return;
        }
    };

    let mut state = crate::sync::engine::SyncState::load(&sync_state_path);
    match rt.block_on(engine.run_once(local)) {
        Ok(merged) => {
            if let Err(e) = merged.write_to_db(conn) {
                tracing::warn!(error = %e, "daemon.sync.write.error");
                return;
            }
            state.last_sync_at = Some(chrono::Utc::now());
            state.last_error = None;
            state.provider = Some(provider_name);
            let _ = state.save(&sync_state_path);
            tracing::info!("daemon.sync.complete");
        }
        Err(e) => {
            tracing::warn!(error = %e, "daemon.sync.error");
            state.last_error = Some(e.to_string());
            let _ = state.save(&sync_state_path);
        }
    }
}
```
